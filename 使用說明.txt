V9.8 手動中心修正版

1. 將本 ZIP 內的 viewer.js 覆蓋 GitHub 根目錄原本的 viewer.js。
2. 在每個場景的 config.json 加入：

"viewTarget": [4.409884, -0.530158, 5.021313]

注意 JSON 前後欄位之間要有逗號。

範例：
{
  "id": "flower",
  "title": "Flower",
  "model": "./scenes/flower/model.ply",
  "sound": "./scenes/flower/sound.mp3",
  "viewTarget": [4.409884, -0.530158, 5.021313]
}

這版修正重點：攝影機初始位置會跟著 viewTarget 一起位移，Reset 與場景切換也會使用各場景自己的中心。
